## Overview

Data display components help you present information in clear, organized, and visually appealing ways.

## Components in this category:

- **Table** - Structured data presentation
- **Card** - Content containers with headers and actions
- **List** - Ordered and unordered content lists
- **Chart** - Data visualization components
- **Progress** - Progress indicators and loading states
- **Badge** - Status indicators and labels
- **Avatar** - User profile images and initials

## Usage Guidelines

1. **Clarity** - Present data in a clear, scannable format
2. **Hierarchy** - Use visual hierarchy to guide attention
3. **Responsive** - Ensure data is readable on all screen sizes
4. **Accessibility** - Provide alternative text and proper labeling

## Best Practices

- Use appropriate chart types for your data
- Provide loading states for data that takes time to load
- Use consistent formatting for similar data types
- Consider data density and readability
