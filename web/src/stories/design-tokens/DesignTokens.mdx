## Colors

Our color palette is designed to be accessible, consistent, and flexible across all themes and contexts.

### Primary Colors
- **Primary**: Used for main actions and brand elements
- **Secondary**: Supporting actions and highlights
- **Accent**: Special emphasis and interactive elements

### Semantic Colors
- **Success**: Positive actions and confirmations
- **Warning**: Caution and attention-grabbing elements
- **Error**: Destructive actions and error states
- **Info**: Informational content and neutral actions

### Neutral Colors
- **Background**: Page and surface backgrounds
- **Surface**: Card and component backgrounds
- **Border**: Dividers and component borders
- **Text**: Content and label text

## Typography

Our typography system ensures readability and hierarchy across all devices and contexts.

### Font Families
- **Primary**: Inter - Clean, modern sans-serif for UI text
- **Monospace**: JetBrains Mono - For code and technical content

### Font Sizes
- **Display**: 48px - Large headings and hero text
- **Heading 1**: 32px - Page titles
- **Heading 2**: 24px - Section titles
- **Heading 3**: 20px - Subsection titles
- **Body Large**: 16px - Important body text
- **Body**: 14px - Standard body text
- **Body Small**: 12px - Supporting text and captions
- **Caption**: 10px - Labels and fine print

## Spacing

Consistent spacing creates visual rhythm and improves readability.

### Spacing Scale
- **xs**: 4px - Tight spacing
- **sm**: 8px - Small spacing
- **md**: 16px - Medium spacing
- **lg**: 24px - Large spacing
- **xl**: 32px - Extra large spacing
- **2xl**: 48px - Section spacing
- **3xl**: 64px - Page spacing

## Border Radius

Rounded corners create a friendly, modern feel while maintaining accessibility.

- **none**: 0px - Sharp corners
- **sm**: 4px - Subtle rounding
- **md**: 8px - Standard rounding
- **lg**: 12px - Prominent rounding
- **xl**: 16px - Strong rounding
- **full**: 50% - Circular elements

## Shadows

Shadows provide depth and hierarchy to our interface.

- **sm**: Subtle elevation
- **md**: Standard elevation
- **lg**: Prominent elevation
- **xl**: Strong elevation

## Breakpoints

Responsive design breakpoints for different screen sizes.

- **xs**: 0px - Mobile phones
- **sm**: 600px - Large phones
- **md**: 900px - Tablets
- **lg**: 1200px - Desktops
- **xl**: 1536px - Large desktops
